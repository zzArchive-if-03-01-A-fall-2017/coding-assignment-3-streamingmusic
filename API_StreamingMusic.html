API

Application programming interface

API is a set of definitions, protocols and tools for building application software.

Its purpose is making it easier for people to use certain technologies in building applications.

SPOTIFY API

In order to be able to use the Spotify Web API, the streaming music service offers

A USER GUIDE
a quick overview of the API to get started

A TUTORIAL
A step-by-step guide to creating the first Web API application

A MANAGER FOR YOUR APPLICATIONS
You can register your app and get keys for authenticating your requests

CODE EXAMPLES
You can get demos, wrappers, and tools for your programming language

Spotify also gives different references to help the developers.

ENDPOINT REFERENCE
You can find there a full documentation of all the endpoints and the data they return.

AUTHORIZATION GUIDE
Here you can read all you need to know about using OAuth 2.0 with the Web API

USING SCOPES
Here you can discover the authorization scopes available to access private information

MIGRATION GUIDE
This helps guide helps you to learn how to update your Metadata API code to use the Web API

Soundcloud API

If you want to become a Soundcloud developer, Soundcloud offers a very detailed API guide about its services.

AUTHENTICATION
Here you can learn more about signing in with Soundcloud, the server-side web applications, mobile & desktop applications and more.

UPLOADING SOUNDS
In this section you can learn how to upload audio files, update metadata, share sounds to social networks and more about sets.

PLAYING SOUNDS
Here you can learn more about the Soundcloud widget and streaming sounds.
